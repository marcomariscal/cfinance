To create an API key

  1. Make sure you already have a Coinbase account

  2. Go to your Coinbase Pro profile To create an API key with your desired permissions (we suggest selecting all permissions for the best experience and to be able to trade freely here

    1. Select "New API Key"
    2. Select desired permissions and retain passphrase (this will be used to sign up here)
    3. Select "Create API Key and retain your "API secret" (this will be used to sign up here)
    4. Your API Key is now ready to be used!
  3. If you have further questions on creating a key, please refer to the documentation here
